**Tool Name**  
AP Stats Score Calculator – A web or app tool estimating AP Statistics exam score (1-5 scale) from raw section inputs. [collegeprep.uworld](https://collegeprep.uworld.com/ap/ap-statistics/score-guide/)

**Purpose**  
Predicts composite score to help students evaluate practice test performance, focus on MCQ vs FRQ weaknesses using equal 50/50 weighting. [albert](https://www.albert.io/blog/ap-statistics-score-calculator/)

**How It Works**  
Sums raw points from 40 MCQ (no penalty) and 6 FRQ (~4 pts each, holistic rubric), weights each section 50%, scales total composite to 1-5 via equating curve approximations. [apcentral.collegeboard](https://apcentral.collegeboard.org/courses/ap-statistics/exam)

**Core Components**  
- Inputs: MCQ correct (0-40), FRQ total (~0-24 pts via 6 questions).  
- Weighting module: 50% MCQ raw, 50% FRQ raw.  
- Curve table: Composite thresholds (e.g., ~70/100 for 4).  
- UI sliders/results with section breakdowns. [revisiondojo](https://www.revisiondojo.com/blog/ap-statistics-2025-exam-format-units-and-scoring-guide-or-revisiondojo)

**Process Flow**  
1. User inputs MCQ correct (0-40), FRQ total points (0-24).  
2. MCQ weighted = (correct/40) * 50.  
3. FRQ weighted = (points/24) * 50.  
4. Composite = MCQ weighted + FRQ weighted (0-100).  
5. Map to AP score using thresholds (e.g., 75+ for 5).  
6. Display score, % per section. [collegeprep.uworld](https://collegeprep.uworld.com/ap/ap-statistics/score-guide/)

**Logic Pattern**  
Equal weighted average: composite = 50*(MCQ_raw/40) + 50*(FRQ_raw/24); tiered lookup on scaled total (curves confidential, approximate from past: ~57-69=3, 70-79=4, 80+=5). [collegevine](https://www.collegevine.com/faq/45923/understanding-the-ap-statistics-grading-scale)

**Key Considerations**  
- FRQ max ~24 pts (E/P/I per part → 0-4 per Q); holistic scoring subjective in practice.  
- Curves vary yearly by difficulty (estimates ±3 pts); no MCQ penalty.  
- Dependencies: Annual rubric updates; limitations: No per-FRQ breakdown; risks: Ignores investigative task depth. [apcentral.collegeboard](https://apcentral.collegeboard.org/media/pdf/ap23-sg-statistics.pdf)

**Inputs & Outputs**  
- **Inputs**: MCQ correct (int 0-40), FRQ total points (int 0-24).  
- **Outputs**: Section weights, composite (0-100), predicted score (1-5), percentile estimate. [albert](https://www.albert.io/blog/ap-statistics-score-calculator/)
