**Tool Name**  
AP World Score Calculator – A web or app tool estimating AP World History: Modern exam score (1-5 scale) from raw section inputs. [collegeprep.uworld](https://collegeprep.uworld.com/ap/ap-world-history-modern/score-guide/)

**Purpose**  
Estimates final AP score to assess student readiness, prioritize study on weak sections like DBQ/LEQ, using official weighting for practice prediction. [albert](https://www.albert.io/blog/ap-world-history-score-calculator/)

**How It Works**  
Aggregates raw points from MCQ, SAQ, DBQ, LEQ; applies section weights (40% MCQ, 20% SAQ, 40% FR total); scales composite to 1-5 via yearly curve approximations. [collegetransitions](https://www.collegetransitions.com/blog/ap-world-history-score-calculator/)

**Core Components**  
- Input fields: MCQ correct (0-55), SAQ total (0-9), DBQ (0-7), LEQ (0-6).  
- Weighting calculator: MCQ/SAQ (Section I 60%), FRQ (Section II 40%).  
- Threshold table for composite-to-score mapping.  
- UI for sliders/results with breakdowns. [collegeprep.uworld](https://collegeprep.uworld.com/ap/ap-world-history-modern/score-guide/)

**Process Flow**  
1. User enters MCQ (0-55), SAQ (0-9), DBQ (0-7), LEQ (0-6).  
2. Section I raw = MCQ + SAQ (max 64) → weighted to 60% of composite.  
3. Section II raw = DBQ + LEQ (max 13) → weighted to 40% of composite.  
4. Composite = (Section I /64 * 60) + (Section II /13 * 40), scaled 0-100.  
5. Map to AP score (e.g., 75+ ≈4-5).  
6. Output score, section %, percentile estimate. [collegetransitions](https://www.collegetransitions.com/blog/ap-world-history-score-calculator/)

**Logic Pattern**  
Weighted sectional sum: composite = 0.6*(MCQ/55 + SAQ/9)/2 + 0.4*(DBQ/7 + LEQ/6)/2; tiered thresholds (vary yearly, e.g., 65-74=3, 75-86=4, 87+=5) from past released data. [collegeprep.uworld](https://collegeprep.uworld.com/ap/ap-world-history-modern/score-guide/)

**Key Considerations**  
- Thresholds exam-year specific (not public; use 2024-2026 averages ±2 pts error); FRQ rubrics (thesis=1, evidence=2-3 pts) subjective in practice.  
- No MCQ guessing penalty; dependencies: Update from College Board samples; limitations: Ignores partial FRQ credit granularity; risks: Overemphasis on raw totals without skill gaps. [apcentral.collegeboard](https://apcentral.collegeboard.org/courses/ap-world-history/exam)

**Inputs & Outputs**  
- **Inputs**: MCQ correct (int 0-55), SAQ points (int 0-9), DBQ points (int 0-7), LEQ points (int 0-6). [collegeprep.uworld](https://collegeprep.uworld.com/ap/ap-world-history-modern/score-guide/)
- **Outputs**: Section weights, composite (0-100), predicted score (1-5), breakdowns/table. [albert](https://www.albert.io/blog/ap-world-history-score-calculator/)
