**Tool Name**  
AP Bio Score Calculator – A web or app tool that estimates a student's AP Biology exam score (1-5 scale) from raw section inputs. [revisiondojo](https://www.revisiondojo.com/blog/ap-biology-score-calculator-how-to-estimate-your-score)

**Purpose**  
Solves the problem of predicting AP exam performance for students practicing or reviewing, using College Board weighting to forecast final scaled scores and guide study focus. [ctrlcalculator](https://ctrlcalculator.com/math/ap-bio-score-calculator/)

**How It Works**  
Accepts user inputs for multiple-choice correct answers and free-response points, computes weighted composite score via scaling formulas, then maps to AP 1-5 scale using annual curve thresholds or approximations. [apcentral.collegeboard](https://apcentral.collegeboard.org/courses/ap-biology/exam)

**Core Components**  
- Input form for MCQ (0-60) and FRQ raw points (0-~34, based on 2 long at 9 pts each + 4 short at 4 pts each). [apcentral.collegeboard](https://apcentral.collegeboard.org/courses/ap-biology/exam)
- Scoring engine with weighting (50% MCQ, 50% FRQ) and scaling logic. [revisiondojo](https://www.revisiondojo.com/blog/ap-biology-score-calculator-how-to-estimate-your-score)
- Lookup table or interpolation for composite-to-AP-score conversion.  
- UI for display and optional percentile/feedback. [calculateconfidence](https://www.calculateconfidence.com/ap-bio-exam-score-calculator/)

**Process Flow**  
1. User enters MCQ correct (e.g., 45/60) and FRQ points (e.g., 25/34). [ctrlcalculator](https://ctrlcalculator.com/math/ap-bio-score-calculator/)
2. Calculate weighted MCQ: (correct / 60) * 50. [revisiondojo](https://www.revisiondojo.com/blog/ap-biology-score-calculator-how-to-estimate-your-score)
3. Calculate weighted FRQ: (points / max_FRQ) * 50. [revisiondojo](https://www.revisiondojo.com/blog/ap-biology-score-calculator-how-to-estimate-your-score)
4. Sum to composite score (0-100).  
5. Map composite to AP score via thresholds (e.g., ~65+ for 4). [collegeprep.uworld](https://collegeprep.uworld.com/ap/ap-biology/score-guide/)
6. Output predicted score with breakdown. [calculateconfidence](https://www.calculateconfidence.com/ap-bio-exam-score-calculator/)

**Logic Pattern**  
Weighted average pattern: composite = (MCQ_raw / 60 * 50) + (FRQ_raw / total_FRQ_pts * 50), followed by equating table lookup (thresholds vary yearly by exam difficulty, approximated from past data). Linear scaling assumes fixed max points; rubric-based FRQ totals dynamic. [ctrlcalculator](https://ctrlcalculator.com/math/ap-bio-score-calculator/)

**Key Considerations**  
- FRQ max points (~34) based on rubrics (9+9+4*4), but varies slightly; use latest CED. [apcentral.collegeboard](https://apcentral.collegeboard.org/courses/ap-biology/exam)
- Exact College Board curves confidential/yearly; approximations from practice data risk ~1-point inaccuracy. [collegeprep.uworld](https://collegeprep.uworld.com/ap/ap-biology/score-guide/)
- No penalties for wrong MCQ; dependencies: accurate past-year tables; limitations: ignores grid-ins if separate; risks: over-reliance without practice context. [study](https://study.com/academy/popular/ap-biology-exam-scoring-information.html)

**Inputs & Outputs**  
- **Inputs**: MCQ correct (integer 0-60), FRQ total points (float 0-34). [ctrlcalculator](https://ctrlcalculator.com/math/ap-bio-score-calculator/)
- **Outputs**: Weighted scores, composite (0-100), predicted AP score (1-5), optional table/percentile. [calculateconfidence](https://www.calculateconfidence.com/ap-bio-exam-score-calculator/)
