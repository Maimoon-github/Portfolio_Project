**Tool Name**
AP Calc BC Score Calculator – A web or app tool estimating AP Calculus BC exam score (1-5 scale) and AB subscore from raw section inputs.[^1][^2]

**Purpose**
Predicts exam performance to help students gauge readiness, identify weak areas, and strategize practice using official weighting.[^4][^1]

**How It Works**
Takes multiple-choice correct answers and free-response points per question, computes weighted composite via scaling (MCQ x1.2 + FRQ total), maps to 1-5 scale using yearly thresholds; separately calculates AB subscore from AB-content subset.[^2][^1]

**Core Components**

- Input fields: MCQ correct (0-45), FRQ1-6 scores (0-9 each).
- Scaling engine: MCQ weighted (max 54), FRQ total (max 54).
- Dual mapping tables: BC composite (0-108) to score; AB subset to subscore.
- UI for sliders, results display, score breakdowns.[^1]

**Process Flow**

1. User inputs MCQ correct (0-45) and FRQ scores (6x 0-9).
2. MCQ weighted = correct × 1.2 (max 54).
3. FRQ total = sum of 6 FRQs (max 54).
4. Composite = MCQ weighted + FRQ total (0-108).
5. Map composite to BC score via thresholds (e.g., 69+ for 5).
6. Compute AB subscore from AB-tagged questions (~70% content).
7. Output BC score, AB subscore, breakdown.[^1]

**Logic Pattern**
Linear weighted sum: composite = (MCQ_correct / 45 × 54) + FRQ_sum, followed by tiered threshold lookup (e.g., 87-108=5, 69-86=4); AB subscore uses parallel calculation on filtered AB items (~27 MCQ + 3-4 FRQ parts).[^2][^1]

**Key Considerations**

- Thresholds vary yearly by exam difficulty (estimates from past data, ±2-3 pts error); FRQ max fixed at 54 but rubric-dependent.
- AB subscore from BC exam AB content only; no guessing penalty on MCQ.
- Dependencies: Update tables annually; limitations: No official curves public; risks: False confidence if ignoring BC-specific topics (series, parametrics).[^1]

**Inputs \& Outputs**

- **Inputs**: MCQ correct (int 0-45), FRQ1-6 (int 0-9 each).
- **Outputs**: Composite score (0-108), predicted BC score (1-5), AB subscore (1-5), section breakdowns/percentiles.[^2][^1]
