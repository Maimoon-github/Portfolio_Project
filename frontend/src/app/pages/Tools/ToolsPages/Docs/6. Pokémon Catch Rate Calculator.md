**Tool Name**  
Pokémon Catch Rate Calculator – A web or app tool computing probability of catching a wild Pokémon based on game mechanics formulas.

**Purpose**  
Estimates capture success rate from Pokémon stats and conditions to optimize Poké Ball choice, HP damage, and status for battling efficiency.

**How It Works**  
Applies generation-specific formulas: modifies base catch rate by HP factor, ball multiplier, status bonus; computes shake probabilities (3-4 checks); outputs % chance via shake escape rates.

**Core Components**  
- Lookup DB: Pokémon catch rates (3-255), ball multipliers (1-5x), status factors (1-2.5x).  
- Formula engine: Gen-specific catch rate/modified rate/shake checks.  
- RNG simulator: Shake probability curves, critical capture.  
- UI selectors/results chart. [smogon](https://www.smogon.com/ingame/guides/capture_mechanics)

**Process Flow**  
1. Select Pokémon, level/HP (current/max), ball type, status (none/sleep/freeze/paralyze).  
2. Compute modified rate a = floor( (3*HPmax - 2*HPcur) * base_rate * ball_mult * status_mult / (3*HPmax) ).  
3. Clamp a to 0-255; if >255, guaranteed catch.  
4. Shake prob per check: floor( (a * 100 / (255 - a/4 +1)) / (next_rnd % 3 +2) ) or similar.  
5. Simulate 3-4 shakes; output % success (e.g., 1-a^4 for Gen III/IV). [bulbapedia.bulbagarden](https://bulbapedia.bulbagarden.net/wiki/Catch_rate)

**Logic Pattern**  
Multiplicative adjustment: a = floor( HP_factor * base * ball * status ); then iterative shake escape: P_shake = f(a); total P_catch = 1 - (1 - P_shake)^shakes; Gen variants (II-IV linear, V+ wobble-based); critical capture ~6.7% bonus.

**Key Considerations**  
- Gen-specific (Gen I simpler, Gen VI+ hex man./critical); exact formulas datamined per game.  
- RNG seed affects sim accuracy; limitations: No shiny/IV inputs; risks: Overestimation ignores flee/bad RNG (±5-10%). [pokemon.fandom](https://pokemon.fandom.com/wiki/Catch_rate)

**Inputs & Outputs**  
- **Inputs**: Pokémon species (base rate), HP cur/max, ball type (mult), status (mult), generation.  
- **Outputs**: Modified rate (0-255), catch % (e.g., 45.2%), shakes needed, critical chance. [dragonflycave](https://www.dragonflycave.com/calculators/gen-iii-iv-catch-rate/)
