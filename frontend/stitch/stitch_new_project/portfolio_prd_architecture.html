# Project: Senior Frontend Architect Portfolio (Maimoon) ## Tech Stack - React 18+ (Functional Components, Hooks) - TypeScript (Strict Mode) - Tailwind CSS (Utility-first, Custom Plugins) - HTML5 / CSS3 (Grid, Flexbox, Animations) - Typography: Space Grotesk ## Visual Identity (Aura-based) - **Primary:** Golden-White (#FFFDF5) - **Secondary:** Radiant Amber-Orange (#FFBF00) - **Tertiary/Protective:** Soft Blue (#E3F2FD / #1976D2) - **Theme:** Professional, Fortunate, Harmonious, Steady ## Site Architecture 1. **Home:** Hero, Projects, Latest Blog, Tools, CTA. 2. **About:** Experience, Skills (HTML/CSS/JS, React), Education. 3. **Portfolio:** Grid/List Filterable View, Individual Case Studies. 4. **Blog:** Index with Search/Filters, Individual Posts. 5. **Tools:** - Directory Overview. - **Hybrid Detail Page:** - Left Sidebar: Interactive Calculator/Tool. - Right Content: Blog-style explanation. 6. **Contact:** Form, Socials. 7. **Utility:** 404, Privacy, Terms, Sitemap. ## Core Principles - Audit before acting. - Precision over verbosity. - Scalability-first. - CRAP Principles (Contrast, Repetition, Alignment, Proximity).
