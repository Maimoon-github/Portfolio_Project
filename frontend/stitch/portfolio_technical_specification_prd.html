# Technical Specification: Professional AI Portfolio (Maimoon) ## 1. Architectural Overview - **Framework:** React-based Single Page Application (SPA) with creative transitions. - **Styling:** Tailwind CSS for responsive utility-first design, preserving pixel-perfect integrity. - **Typography:** Space Grotesk (Primary/Variable) - Geometric, modern, and efficient. - **Core Principles:** CRAP (Contrast, Repetition, Alignment, Proximity), Efficiency, and Business Impact. ## 2. Visual Identity (Aura-Based Palette) - **Primary (Core):** Golden-White (`#FFFDF0`) - Radiant, prosperous, clean. - **Accent 1 (Energy):** Amber-Orange (`#FFBF00`) - Creativity, manifestation, and CTA highlights. - **Accent 2 (Guard):** Protective Blue (`#4A90E2`) - Stability, trust, and depth. - **Borders/Neutral:** Deep Moss (`#1A2421`) - Grounding elements, skills, and cards. - **Highlights:** Neo-Mint (`#98FF98`) - High-conversion interactivity. ## 3. Layout Strategy: The "Combo Layout" - **Navigation:** Fixed header with frictionless flow. - **Structure:** - **Hero:** Aura-focused introduction (Golden-White/Amber) with clear value prop. - **Projects:** Grid-based cards with Deep Moss borders; responsive adaptability. - **Content/Blog:** Minimalist focus with creative whitespace to reduce cognitive load. - **Live Demos/Workflows:** SPA-style asynchronous loading for MLOps pipelines. ## 4. Engineering Constraints - **Audit-First:** Analyze component hierarchy before scaling. - **Precision:** Zero dead code; justified dependencies only. - **Responsive Integrity:** Desktop-first logic with mobile-first media query adaptation. - **Performance:** Minimized bounce rates via optimized load times and frictionless transitions.
