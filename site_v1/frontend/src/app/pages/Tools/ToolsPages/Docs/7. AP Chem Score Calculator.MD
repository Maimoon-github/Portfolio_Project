**Tool Name**  
AP Chem Score Calculator – A web or app tool estimating AP Chemistry exam score (1-5 scale) from raw section inputs. [collegevine](https://www.collegevine.com/faq/55543/ap-chemistry-exam-grading-scale-system)

**Purpose**  
Predicts performance on practice exams to target MCQ/FRQ weaknesses, using 50/50 weighting for study prioritization. [collegevine](https://www.collegevine.com/faq/51169/ap-chemistry-grading-scale-how-does-it-work)

**How It Works**  
Sums MCQ correct (0-60) and FRQ points (0-~46 from 3 long/4 short), weights each 50% of composite, maps via yearly curve to 1-5 scale. [blog.prepscholar](https://blog.prepscholar.com/ap-chemistry-exam)

**Core Components**  
- Inputs: MCQ (0-60), FRQ total (~46 pts: 10/pt long, 4/pt short).  
- Scaling: MCQ/60*50 + FRQ/46*50.  
- Threshold lookup table (e.g., 75+ for 5).  
- UI for entry/results breakdown. [collegevine](https://www.collegevine.com/faq/55543/ap-chemistry-exam-grading-scale-system)

**Process Flow**  
1. User enters MCQ correct (0-60), FRQ points (0-46).  
2. MCQ weighted = (correct / 60) × 50.  
3. FRQ weighted = (points / 46) × 50.  
4. Composite = MCQ + FRQ weighted (0-100).  
5. Map to AP score (e.g., 70-100=5).  
6. Output score with % per section. [collegevine](https://www.collegevine.com/faq/51169/ap-chemistry-grading-scale-how-does-it-work)

**Logic Pattern**  
Equal weighted sum: composite = 50*(MCQ_raw/60) + 50*(FRQ_raw/46); tiered thresholds from past curves (~57=3, 70=4, 80+=5, varies yearly). [blog.prepscholar](https://blog.prepscholar.com/ap-chemistry-exam)

**Key Considerations**  
- FRQ max ~46 pts rubric-dependent (partial credit per part); no MCQ penalty.  
- Curves confidential/yearly (±3 pts error in estimates); dependencies: Update from CB samples; risks: Subjective FRQ self-score variance. [study](https://study.com/academy/popular/ap-chemistry-exam-scoring-information.html)

**Inputs & Outputs**  
- **Inputs**: MCQ correct (int 0-60), FRQ total (int 0-46).  
- **Outputs**: Composite (0-100), predicted score (1-5), section %. [collegevine](https://www.collegevine.com/faq/55543/ap-chemistry-exam-grading-scale-system)
