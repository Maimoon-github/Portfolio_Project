**Tool Name**  
AP Physics 1 Score Calculator – A web or app tool estimating AP Physics 1 exam score (1-5 scale) from raw section inputs. [collegeprep.uworld](https://collegeprep.uworld.com/ap/ap-physics-1/score-guide/)

**Purpose**  
Predicts composite score from practice tests to identify MCQ/FRQ gaps, aiding study focus with 50/50 section weighting. [collegeprep.uworld](https://collegeprep.uworld.com/ap/ap-physics-1/score-guide/)

**How It Works**  
Combines MCQ correct (0-50) and FRQ points (0-~50 from 5 questions), weights each 50%, scales composite via yearly thresholds to 1-5. [num8ers](https://num8ers.com/score-calculator/ap-physics-1-score-calculator/)

**Core Components**  
- Inputs: MCQ (0-50), FRQ total (~50 pts: exp design 12, translation 12, short ans 7x3).  
- Weighting: MCQ/50*50 + FRQ/50*50.  
- Threshold table (e.g., 70+=5).  
- UI sliders/breakdown display. [num8ers](https://num8ers.com/score-calculator/ap-physics-1-score-calculator/)

**Process Flow**  
1. User enters MCQ correct (0-50), FRQ points (0-50).  
2. MCQ weighted = (correct / 50) × 50.  
3. FRQ weighted = (points / 50) × 50.  
4. Composite = sum (0-100).  
5. Map to score (e.g., 40-53=3).  
6. Output with section %. [num8ers](https://num8ers.com/score-calculator/ap-physics-1-score-calculator/)

**Logic Pattern**  
Equal weighted average: composite = 50*(MCQ_raw/50) + 50*(FRQ_raw/50); tiered lookup (~25-39=2, 40-53=3, 54-69=4, 70+=5 from past data). [collegeprep.uworld](https://collegeprep.uworld.com/ap/ap-physics-1/score-guide/)

**Key Considerations**  
- FRQ max ~50 pts rubric-based (partial credit); no MCQ penalty.  
- Thresholds vary yearly (±2 pts); update from CB data; limitations: No per-FRQ detail; risks: Self-scoring FRQ bias. [collegeprep.uworld](https://collegeprep.uworld.com/ap/ap-physics-1/score-guide/)

**Inputs & Outputs**  
- **Inputs**: MCQ correct (int 0-50), FRQ total (int 0-50).  
- **Outputs**: Composite (0-100), predicted score (1-5), section weights. [num8ers](https://num8ers.com/score-calculator/ap-physics-1-score-calculator/)
